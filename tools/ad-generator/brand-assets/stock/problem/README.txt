problem klasörü
================

Bu klasöre problem sahnesi için kullanılmasını istediğin Pexels (veya başka kaynaklı) mp4 dosyalarını koy.

Dosya isimlendirme: kebab-case + içerik açıklayıcı. Pipeline filename'den ne olduğunu çıkarır.
Örnek isimler:
  - frustrated-athlete-mirror.mp4
  - injured-runner-knee-pain.mp4
  - bad-squat-form-side.mp4
  - confused-gym-decision.mp4

Problem = sorun anı. Yorgun, çaresiz, yanlış form gibi negatif duygular.
